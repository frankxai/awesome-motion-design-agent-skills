# Contributing

Add resources that improve real shipped motion quality for AI-assisted design and frontend work.

Good additions include:

- Official docs.
- Reusable skills.
- MCP servers with clear scopes.
- Motion examples with source code.
- QA rubrics.
- Export/capture recipes.

Avoid:

- Giant unsorted link dumps.
- Unlicensed animation assets.
- Prompt packs with no visible output quality.
- Unreviewed MCP servers with broad permissions.
